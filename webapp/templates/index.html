Filtros Implementados

8 filtros espaciais para redução de ruído

Filtro de Média

Substitui cada pixel pela média dos vizinhos. Eficaz para ruído gaussiano, mas pode borrar bordas.

Kernel 3×3 Kernel 7×7

Filtro Gaussiano

Média ponderada usando distribuição gaussiana. Suaviza mantendo mais detalhes que o filtro de média.

Kernel 3×3 Kernel 7×7

Filtro de Mediana

Substitui pelo valor mediano dos vizinhos. Excelente para ruído sal e pimenta, preserva bordas.

Kernel 3×3 Kernel 7×7

Filtro de Moda

Substitui pelo valor mais frequente. Útil para imagens com padrões repetitivos.

Kernel 3×3 Kernel 7×7
